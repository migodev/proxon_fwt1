# Proxon FWT1 for Symcon

Folgende Module beinhaltet das Proxon FWT1 for Symcon Repository:

- __Proxon FWT1__ ([Dokumentation](https://github.com/migodev/proxon_fwt1/tree/main/Proxon%20Panel))  
	Das Modul erlaubt es, eine EInzelraumsteuerung auf Basis der Zimmermann PROXON FrischluftWärmetechnik Anlage 1.0 via Modbus TCP zu implementieren.
    Die PROXON Anlage muss dazu bereits per Modbus TCP angebunden sein.
